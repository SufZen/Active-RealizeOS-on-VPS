# Holdings Normalization

Use this directory for normalized holdings snapshots and source notes.

## Preferred Fields

- `as_of`
- `account_id`
- `asset_id`
- `asset_type`
- `quantity`
- `cost_basis`
- `market_value`
- `currency`
- `source_name`
- `source_confidence`

## V1 Rule

If the source is incomplete or stale, record the gap before using the data for any recommendation.
