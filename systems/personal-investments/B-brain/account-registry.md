# Account Registry

## Purpose

Defines the source of truth for all personal investment accounts and sleeves.

## Sources

- Manual portfolio spreadsheet
- Broker or exchange exports when available
- Private deal notes and manually maintained cap table / ownership records
- Real estate and off-platform allocations tracked manually

## Rules

- Every account must declare whether it is live, paper, or manual.
- Manual assets are valid in v1, but must be tagged with update date and confidence.
- Recommendations must flag any account that has not been refreshed recently.
