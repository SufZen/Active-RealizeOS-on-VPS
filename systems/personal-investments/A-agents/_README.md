# Personal Investments Agent Team

## Core Team

| Agent | File | Scope |
|-------|------|-------|
| Orchestrator | `orchestrator.md` | Entry point, routing, cross-skill coordination |
| Investment CIO | `investment-cio.md` | Strategy, allocation, thesis maintenance |
| Risk Controller | `risk-controller.md` | Guardrails, vetoes, risk framing |
| Execution Controller | `execution-controller.md` | Approval queue, paper-trade packets, audit formatting |

## Routing Guide

| Request Type | Route |
|-------------|-------|
| General investment help, dashboard, coordination | Orchestrator |
| Portfolio strategy, allocation, thesis, market view | Investment CIO |
| Risk questions, exposure, constraints, veto checks | Risk Controller |
| Approval queue, paper trade, audit trail, sync process | Execution Controller |

## Team Rule

The CIO may recommend. The risk controller may block. The execution controller may package and queue. No one may bypass the active authority mode.
