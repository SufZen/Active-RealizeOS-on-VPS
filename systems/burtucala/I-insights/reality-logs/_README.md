# Reality Logs — Capture Process

> Raw field notes for content creation. These are the source material for Reality Log (Format 1) and Case Files (Format 5).

## How to Capture

After any notable field experience (site visit, meeting, bureaucratic encounter, client interaction):

1. Write a 3-5 sentence note within 24 hours
2. Save as `[YYYY-MM-DD]-[slug].md` in this directory
3. Include: what happened, what surprised you, what you learned
4. Tag with likely pillar (P1/P2/P3/P4)

## File Template

```markdown
# [Short Title]

**Date:** YYYY-MM-DD
**Pillar:** P1 / P2 / P3 / P4
**Location:** [Where it happened]

## What Happened
[2-3 sentences about the event]

## What Surprised Me
[1-2 sentences about what was unexpected]

## What I Learned
[1-2 sentences about the takeaway]
```

## How These Get Used

- Review weekly during content planning
- Reality Logs (Format 1) draw directly from these notes
- Case Files (Format 5) may combine multiple reality logs
- The Writer references this directory before drafting field-based content
