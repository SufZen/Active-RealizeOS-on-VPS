# Company Investments Agent Team

## Core Team

| Agent | File | Scope |
|-------|------|-------|
| Orchestrator | `orchestrator.md` | Entry point, routing, cross-skill coordination |
| Investment CIO | `investment-cio.md` | Treasury strategy, allocation, thesis maintenance |
| Risk Controller | `risk-controller.md` | Guardrails, vetoes, governance checks |
| Execution Controller | `execution-controller.md` | Approval queue, paper-trade packets, audit formatting |

## Team Rule

Company investing must be more governance-heavy than personal investing. Any ambiguity about mandate, liquidity, or authority defaults to a block.
