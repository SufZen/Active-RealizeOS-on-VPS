# Method: Decision Framework

## Purpose
Make better decisions faster by structuring the thought process.

## When to Use
When facing a significant business decision with multiple options and unclear trade-offs.

## Steps

### Step 0: The Realization Filter (Gate Check)

Before entering the full decision framework, run the 3-question filter. If any answer is "no," strongly consider passing before investing more energy:

1. **Potential Test:** Is there a massive gap between the current wasted state and the maximum potential we can create from it?
2. **Agenda Test:** Does this action constitute optimization and value creation (urban healing, systemic improvement), or does it harm the environment or add no meaningful value?
3. **Role Test:** Can the Architect build a pipe here and hand it off, or does this pull into gray operational work that drains learning energy?

If all three pass, proceed to the full framework below.

---

1. **Define the decision** — State it as a clear question: "Should we [X] or [Y]?"
2. **List constraints** — What's non-negotiable? (Budget, timeline, team capacity, values)
3. **Generate options** — At least 3 options (including "do nothing")
4. **Score each option** on these dimensions (1-5 scale):
   | Criteria | Weight | Option A | Option B | Option C |
   |----------|--------|----------|----------|----------|
   | Revenue impact | 5 | ... | ... | ... |
   | Time to execute | 4 | ... | ... | ... |
   | Risk level | 3 | ... | ... | ... |
   | Alignment with values | 4 | ... | ... | ... |
   | Reversibility | 3 | ... | ... | ... |
5. **Calculate weighted scores** — Multiply each score by weight, sum per option
6. **Gut check** — Does the winning option feel right? If not, explore why.
7. **Decide and document** — Record the decision, reasoning, and what would make you revisit it

## Expected Output
A decision matrix with clear recommendation and documented reasoning.

## Notes
- The framework doesn't make the decision — it reveals your thinking
- If the gut check disagrees with the matrix, trust the gut and investigate what criteria you're missing
- Record every significant decision — future-you will thank you
- Set a "revisit by" date for important decisions
